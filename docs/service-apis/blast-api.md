# Blast Chain

Blast is the only Ethereum L2 with native yield for ETH and stablecoins. Blast yield comes from ETH staking and RWA protocols. The yield from these decentralized protocols is passed back to Blast users automatically. The default interest rate on other L2s is 0%. On Blast, itâ€™s 4% for ETH and 5% for stablecoins.
