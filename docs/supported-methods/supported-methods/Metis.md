# Metis

Metis is a Layer 2 Ethereum scaling solution aiming to make building Decentralized Autonomous Companies (DACs) easy and cheap. By leveraging the Metis supported methods, developers can harness this scalable infrastructure with actions such as querying balances, executing transactions, and interacting with smart contracts on the Metis network.

Metis's focus on scalability and cost-effectiveness makes it an appealing choice for a wide range of decentralized applications and enterprises.

- [Metis Developer Documentation](https://docs.metis.io)
- [Metis Community Forum](https://forum.metis.io)
- [Metis Github Repository](https://github.com/metisdao)

The following methods are supported on the Metis Mainnet:
