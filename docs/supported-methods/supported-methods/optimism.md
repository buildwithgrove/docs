# Optimism

Optimism is an Ethereum Layer 2 scaling solution, designed to enhance transaction speeds and reduce costs while maintaining close alignment with the Ethereum mainnet. Optimism's Supported Methods provide developers with a diverse array of functionalities, facilitating interactions such as balance queries, transaction executions, and smart contract integrations on the Optimism network.

- [Optimism Developer Documentation](https://docs.optimism.io/)
- [Optimism Community Forum](https://community.optimism.io/)
- [Optimism Github Repository](https://github.com/optimism-io/optimism)

---

Grove's integration with Optimism streamlines access to its Layer 2 scaling features, enabling developers to efficiently use the Optimism network within their projects.

## Supported Methods

The following methods are available on the Optimism Mainnet:

|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------|

| `eth_call`                             | Call any read-only function on a deployed contract                                                                                                              | [View method](#)  |
| `eth_chainId`                          | Returns the current network/chain ID, used to sign replay protected transaction introduced in EIP155                                                           | [View method](#)  |
| `eth_createAccessList`                 | Creates an EIP-2930 access list that you can include in a transaction                                                                                          | [View method](#)  |
| `eth_estimateGas`                      | Generates and returns an estimate of how much gas is necessary to allow the transaction to complete. The estimate is likely significantly more than the actual cost, for a variety of reasons | [View method](#) |

| `eth_gasPrice`                         | Returns current gas price in wei                                                                                                                               | [View method](#)  |

| `eth_getBlockbyNumber`                 | Returns information about a block by block number                                                                                                              | [View method](#)  |
