# OKC Chain

OKC Chain, developed by OKCoin, is an advanced public blockchain platform. It is specifically designed to support a wide array of decentralized financial applications, utilizing OKCoin's extensive expertise in financial technology and blockchain. This platform offers a secure, efficient, and scalable infrastructure for decentralized exchange and value transfer across its network.

OKC Chain's supported methods provide developers with a diverse set of tools for effective interaction with the chain. These methods encompass various functionalities, such as querying account balances, processing transactions, estimating gas costs, and more, thus offering a comprehensive suite of tools for developing and managing decentralized applications.

- [OKC Chain Developer Documentation](https://docs.okcchain.blockchain/)
- [OKC Chain Community Forum](https://community.okcchain.blockchain/)
- [OKC Chain Github Repository](https://github.com/okcchain/blockchain)

The following methods are available on the OKC Mainnet:

|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------|
